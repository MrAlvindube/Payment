API endpoints

core routes

Use these endpoints to power the experience:

  • POST /payments — create a payment
  • GET /payments — list payments
  • POST /payments/{id}/authorize — hold funds
  • POST /payments/{id}/capture — finalize collection
  • POST /payments/{id}/refund — return funds
